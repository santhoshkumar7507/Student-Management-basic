# 🎓 Student Management System (Basic)

Welcome to the **Student Management System**! A simple, terminal-based Python application designed to help you organize student records seamlessly.

---

## 🚀 Features

- **Add Students**: Register new students with their ID, name, age, and course information.
- **View Records**: Display a properly formatted list of all registered students.
- **Search**: Quickly find specific students by entering their unique Student ID.
- **Update Records**: Modify existing student particulars exactly when you need to.
- **Delete Records**: Remove stale or incorrect student entries safely.

## 🛠️ Built With

- **[Python](https://www.python.org/)** - Core programming language used for logic and flow.

## ⚙️ Getting Started

### Prerequisites
Make sure you have Python installed on your system. You can download the latest version from [here](https://www.python.org/downloads/).

### Installation & Execution

1. Clone this repository:
   ```bash
   git clone https://github.com/santhoshkumar7507/Student-Management-basic.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Student-Management-basic
   ```
3. Run the application:
   ```bash
   python main.py
   ```
